READ ME!

Step 1
Extract the VILLAREALJESSIE.zip

Step 2 
Install the Visual Studio (Community or Professional)

Step 3 
Install the XAMPP, then open the XAMPP and start the Apache and MySQL then click the admin in the MySQL

Step 4
Create database and name it "villareal" then click the "villareal" and import the "villareal"

Step 5
After importing the database click the "villareal" folder and open the VILLAREALJESSIE.sln

Step 6 
If you already done this step 1-5 then you can run the program.
